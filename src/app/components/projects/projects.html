Sistema de Gestión de Inventario

Aplicación web para administrar el inventario de materia prima, registrar entradas y salidas, generar alertas por niveles críticos y visualizar reportes estadísticos. Incluye un frontend moderno y un backend Node.js que actúa como proxy hacia una API externa.

Tecnologías: Angular, Node.js, Docker, SonarCloud, AWS, PostgreSQL, Spring Boot

Space Shooter

Videojuego 2D estilo arcade desarrollado en JavaScript, donde el jugador pilota una nave espacial y enfrenta oleadas de enemigos con distintos patrones de ataque. Incluye sistema de puntuación, colisiones, animaciones y mejoras visuales.

Tecnologías: HTML5 Canvas, JavaScript (ES6), CSS, Vite
